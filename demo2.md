# 链接
- 外部链接  
    - 语法：\[链接名称](URL)
    - 例子：*\[百度首页](https://www.baidu.com)*  效果：[百度首页](https://www.baidu.com)

- 内部链接
    - 语法：\[链接名称](本地路径)
    - 同仓库另一md文档：*\[demo](demo.md)*  效果：[同仓库的demo1.md](demo1.md)
    - 同md文档其他标题：*\[跳转本文其他部分](demo.md#标题)*  效果：[跳转代码块](demo2.md#代码块)
    
- 引用式链接
    - 语法：在文档内部定义，直接引用。
    - 引用链接方式：[baidu]
